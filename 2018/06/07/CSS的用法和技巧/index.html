  • 在其父元素的css中加入clearfix：：after指令，即可实现元素的左右布局。

如何实现左中右布局

  • 对子元素加入float：left；指令
  • 在其父元素中加入clearfix：：after指令
  • 在子元素之间添加长度合适的margin属性
  • 使用：nth-child（）伪元素将第一个元素的margin设为0
  • 将父元素的display值定为inline-block即可实现左中右布局
